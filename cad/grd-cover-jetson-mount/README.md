# GRD Cover — Custom Redesign

A redesigned replacement cover for the Wave Rover's General Driver Board (GRD) compartment.
The original cover is a flat plastic sheet that sits over the electronics bay.
This custom version is functionally identical but adds several modifications
required for the Jetson Orin Nano Super integration.

## Modifications from Original

- Additional opening for access to the outer 40-pin header on the General Driver Board.
- Enlarged OLED window for better visibility.
- Cable routing channels for the Jetson WiFi antenna wires, allowing them to
  pass through to the rear of the rover, where the antennas mount externally.
